## Featured Projects

### niftyConnect

A notification automation plugin for WordPress and WooCommerce. It helps site owners stay informed through customizable email and Telegram alerts.

- Supports posts, comments, user activity, orders, and stock events
- Provides custom templates, recipient routing, and delivery controls
- Built with an extensible provider system for adding notification channels

**Tech:** `PHP` `TypeScript` `WordPress` `WooCommerce` `Telegram Bot API`

[View Live Plugin](https://wordpress.org/plugins/niftyconnect/) · [Source Code](https://github.com/juah255/niftyconnect)

- Covers REST APIs, validation, authentication, and authorization

**Tech:** `TypeScript` `NestJS` `PostgreSQL` `Prisma` `Docker`

A full-stack learning platform designed around the needs of both students and instructors.

- Includes JWT authentication and role-based access control
- Combines a FastAPI backend with a responsive Next.js frontend
